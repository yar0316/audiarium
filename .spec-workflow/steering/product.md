# Product Overview

## Product Purpose

音楽ファイル（MP3）を一元管理し、Web・モバイル両方で再生を楽しめる個人向け音楽プレイヤーアプリ。ローカルに散らばった音楽ファイルを整理し、どのデバイスからでもアクセス・再生できる環境を提供する。

## Target Users

**プライマリユーザー**: 開発者本人（個人利用）

**ニーズ・ペインポイント**:

- 複数デバイス間で音楽ライブラリを同期したい
- ストリーミングサービスにない楽曲を管理したい
- シンプルで軽量な音楽プレイヤーが欲しい
- 自分好みにカスタマイズできる環境が欲しい

## Key Features

1. **音楽ファイル管理**: MP3ファイルのアップロード、メタデータ編集、削除
2. **プレイリスト機能**: 楽曲をグループ化し、再生順を管理
3. **音楽再生**: Web/モバイル両対応のストリーミング再生
4. **検索・フィルター**: タイトル、アーティスト、アルバムでの検索
5. **クロスプラットフォーム**: Web (Next.js) + モバイル (Flutter)

## Business Objectives

- 個人の音楽ライブラリを一元管理する
- どのデバイスからでも音楽にアクセスできるようにする
- 開発スキル向上（Next.js API Routes、Flutter、Supabase）

## Success Metrics

- **楽曲登録数**: 個人ライブラリの全楽曲を登録
- **日常利用**: メインの音楽プレイヤーとして使用
- **安定性**: エラーなく再生できる

## Product Principles

1. **シンプルさ優先**: 必要最小限の機能から始め、複雑化を避ける
2. **個人最適化**: 汎用性より自分の使いやすさを重視
3. **段階的開発**: Web版を完成させてからモバイル版に着手

## Monitoring & Visibility (if applicable)

- **Dashboard Type**: Web-based（Next.js）
- **Real-time Updates**: 不要（個人利用のため）
- **Key Metrics Displayed**: 楽曲数、プレイリスト数、最近再生した曲
- **Sharing Capabilities**: 不要（個人利用）

## Future Vision

### Potential Enhancements

- **歌詞表示**: 再生中の楽曲に歌詞を表示
- **イコライザー**: 音質調整機能
- **オフライン再生**: モバイルでのダウンロード再生
- **スマートプレイリスト**: 再生履歴に基づく自動プレイリスト生成
